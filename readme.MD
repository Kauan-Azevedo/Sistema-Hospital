### Compilar o arquivo 🌟

- Compilar arquivo:

    ```bash
      gcc main.c -L/usr/lib64/mysql -lmysqlclient -o bin/main.exe
    ```

- Executar programa compilado:

    ```bash
      ./bin/main.exe
    ```

- (Ubuntu): Iniciar servico mysql:

  ```bash
    sudo service mysql start
  ```

- (Ubuntu): Iniciar mysql:

  ```bash
    sudo mysql -p
  ```
